gnns4hri is an umbrella for a number of projects where structured machine learning is applied to Human-Robot Interaction.


The following is a list of the current projects:

  • Sonata: A tool to generate HRI datasets.
  • SNGNN1D: A GNN model which estimates the navigation social compliance value for a particular scene.
  • Datasets: A collection of structured datasets related to social navigation.
